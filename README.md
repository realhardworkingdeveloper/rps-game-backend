## Description

Degen Games provides live PvP gambling games on Solana Blockchain. This repo contains Backend on Nest.JS for this project

## Stack
- Nest.JS
- TypeScript
- MongoDB
- Redis
- Socket.IO

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start:dev

# production mode
$ yarn build && yarn start:prod
```
